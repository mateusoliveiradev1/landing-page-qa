# 🎯 Landing Page QA Agent
Stop wasting thousands of dollars sending paid traffic to broken landing pages. 
This autonomous agent acts as your elite QA Engineer. It actively crawls your pages, tests forms, verifies UTM parameters, and captures **Visual Proofs** (screenshots) of layout breaks before you launch your campaigns.

### ✨ Enterprise Features
* **Visual Evidence:** Captures real browser screenshots of errors.
* **Chain-of-Thought Validation:** Thinks like a human QA to find edge cases.
* **Tracking Audit:** Verifies GTM and UTM presence automatically.
